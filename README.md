# Let'CourtSide

Let'CourtSide is a mobile-first web app that helps tennis fans see who is attending tournaments and connect with others in the same sessions.

## Why I built this
Attending tennis tournaments can be a social experience, but it’s often hard to know who else is going or open to meeting. I built Let'CourtSide to make those connections easier and more natural.

## Features
- Browse upcoming ATP and WTA tournaments
- Indicate which sessions you're attending
- See how many others are going
- Opt in to meet other fans

## Tech Stack
- Flask (Python)
- SQLAlchemy
- PostgreSQL / SQLite
- Jinja2 templates
- SendGrid integration
