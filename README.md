# Let’CourtSide

Let’CourtSide is a mobile-first social platform that helps tennis fans see who is attending tournaments and connect with others in the same sessions.

## Why I built this
Attending tennis tournaments is often a shared experience, but it’s surprisingly hard to know who else is going or open to meeting. I built Let’CourtSide to make those connections more natural by giving fans a simple way to signal attendance and find others in the same moment.

## What it does
- Browse upcoming ATP and WTA tournaments
- Select specific days and sessions you’re attending
- See how many others are going
- Opt in to meet other fans at the event

## How it works
1. Create a profile
2. Select the tournaments and sessions you’re attending
3. Choose whether you’re open to meeting others
4. View who else is going and connect at the event

## Tech Stack
- Flask
- Python
- SQLAlchemy
- PostgreSQL / SQLite
- Jinja2
- SendGrid (email notifications)

## Status
Actively building and iterating

## Vision
Let’CourtSide is built around a simple idea: the best parts of attending live sports are often the people you meet. The goal is to remove friction between fans who are already in the same place at the same time and make those connections easier to happen.
