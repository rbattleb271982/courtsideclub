✅ REPLIT AGENT PROMPT — Add Modal Welcome Message
Replace the current green welcome banner on /my-tournaments with a modal popup.
Here’s what to do:
In templates/user/my_tournaments.html:
Remove the green welcome banner (the one with “Welcome back to CourtSide Club…”)
Add the following HTML just before the closing </body> tag:
<!-- Welcome Modal -->
<div id="welcomeModal" class="modal-overlay">
  <div class="modal-box">
    <h2>Welcome to CourtSide Club — Your Lounge Awaits.</h2>
    <p>Check out the tournaments you’re attending, and see who else is going!</p>
    <button id="closeModalBtn">Let’s Go</button>
  </div>
</div>
In your main CSS file (or <style> block in the same template), add this:
.modal-overlay {
  display: none;
  position: fixed;
  top: 0; left: 0;
  width: 100%; height: 100%;
  background-color: rgba(0,0,0,0.5);
  z-index: 1000;
  justify-content: center;
  align-items: center;
}

.modal-box {
  background: white;
  padding: 2rem;
  border-radius: 1rem;
  text-align: center;
  box-shadow: 0 4px 15px rgba(0,0,0,0.2);
  max-width: 90%;
}

.modal-box h2 {
  font-size: 1.5rem;
  margin-bottom: 0.5rem;
}

.modal-box p {
  margin-bottom: 1rem;
  font-size: 1rem;
}

.modal-box button {
  background-color: #2c7a7b;
  color: white;
  padding: 0.5rem 1.5rem;
  border: none;
  border-radius: 0.5rem;
  cursor: pointer;
  font-weight: bold;
}
Add the following JavaScript just before </body> in the same HTML file:
<script>
  document.addEventListener("DOMContentLoaded", function () {
    const modal = document.getElementById("welcomeModal");
    const closeBtn = document.getElementById("closeModalBtn");

    if (!sessionStorage.getItem("welcomeModalShown")) {
      modal.style.display = "flex";
    }

    closeBtn.addEventListener("click", function () {
      modal.style.display = "none";
      sessionStorage.setItem("welcomeModalShown", "true");
    });
  });
</script>
⚠️ Use only one checkpoint if needed.
✅ Do not finish until you’ve fully tested that:
The modal appears on first visit to /my-tournaments after login
It disappears when clicking “Let’s Go”
It doesn’t reappear on refresh during the same session
