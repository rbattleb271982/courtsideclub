Implement Password Reset Request Flow in Flask
Context:
CourtSide Club is a premium tennis fan community platform. We want to add a secure password reset request flow so users can request a password reset link by submitting their email. The system will generate a secure token valid for 1 hour, send a branded password reset email containing the token link, and redirect the user with a success message.
Requirements:
Create a new Flask route at /reset_password/request supporting GET and POST.
On GET, render a reset_password_request.html template with an email input form styled consistently with CourtSide Club branding.
On POST, accept the email, lookup the user in the database. If found:
Generate a secure token with itsdangerous.URLSafeTimedSerializer, using app secret key and a salt 'password-reset-salt'.
Generate a reset URL for /reset_password/confirm/<token> (external URL).
Send a password reset email using existing send_password_reset_email(to_email, first_name, reset_url) function.
Regardless of whether the email exists, redirect back to the login page with a flash message: "If your email is registered, you will receive a password reset link shortly."
Ensure email sending uses the new premium HTML password reset email design.
Protect the route against common abuse by adding a rate limiter allowing max 5 requests per hour per IP (use flask-limiter or equivalent).
Add unit tests to verify:
Token generation and expiration
Email sending is triggered on valid requests
Flash message appears regardless of user existence
The solution should use itsdangerous for token generation and Flask flash for messages.
Deliverables:
Updated routes/auth.py with /reset_password/request route (GET and POST)
New templates/reset_password_request.html (styled form with email input and submit button)
Update or confirm existing services/email.py contains send_password_reset_email function
Example unit tests for the route in tests/test_auth.py
Additional Info:
Use app config SECRET_KEY for token generation
Token expires after 3600 seconds (1 hour)
Use url_for with _external=True to generate reset link
Make sure all UI messages are clear and professional, aligned with CourtSide Club brand voice
Example snippet for token generation and email sending (to include in the route):
from itsdangerous import URLSafeTimedSerializer
from flask import current_app, url_for

serializer = URLSafeTimedSerializer(current_app.config['SECRET_KEY'])

token = serializer.dumps(user.email, salt='password-reset-salt')
reset_url = url_for('auth.reset_password_confirm', token=token, _external=True)
send_password_reset_email(user.email, user.first_name, reset_url)
