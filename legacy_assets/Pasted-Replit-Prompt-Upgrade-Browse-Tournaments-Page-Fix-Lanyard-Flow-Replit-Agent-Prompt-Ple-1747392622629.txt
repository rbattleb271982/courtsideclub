🔧 6. Add Conditional Logic to Lanyard Page
In routes/user.py (or equivalent route file), update the /lanyard route:

@user_bp.route("/lanyard")
@login_required
def lanyard():
    selected = UserTournament.query.filter_by(user_id=current_user.id, attending=True).first()

    if not selected or not selected.sessions:
        flash("In order to order your lanyard, you have to select a tournament, day, and session first.", "warning")
        return redirect(url_for("user.my_tournaments"))

    return render_template("user/lanyard.html", selected=selected)
✅ This ensures that users must select a tournament + sessions before they can order.
